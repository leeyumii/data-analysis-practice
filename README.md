# data-analysis-practice

#프로젝트 목적
본 프로젝트는 데이터 분석 수업에서 진행한
기본 시각화 실습 결과를 정리하기 위해 수행되었다.

#사용 데이터
서울특별시 공공자전거 일별 대여건수

#분석 및 시각화 내용
- 날짜별 따릉이 대여건수 변화(선그래프)
- 따릉이 일별 대여건수 분포(히스토그램)
- 요일별 평균 따릉이 대여건수(막대그래프)

#사용 도구
-Python
-Pandas
-matplotlib
-seaborn

#느낀 점
그래프를 단순히 그리는 것이 아니라,
분석 목적에 맞는 그래프 선택이 중요하다는 것을 알게 되었다.

#결과요약
1. 계절 및 기온에 따른 뚜렷한 감소 추세
그래프의 하반기 흐름을 보면, 9월과 10월 사이에 가장 높고 11월부터 1월로 갈수록 대여 건수가 급격히 하락하는 양상을 보입니다. 이는 기온이 낮아지는 겨울철에 야외 활동인 자전거 이용 수요가 눈에 띄게 줄어든다는 점을 잘 보여줍니다.

2. 주기적인 급락 패턴 (요일 영향)
주말/평일 차이: 미세하게 반복되는 파동은 출퇴근 수요가 높은 평일과 이용 패턴이 다른 주말의 특성이 반영된 것으로 보입니다.
